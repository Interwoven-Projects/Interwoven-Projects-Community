# Labels & Tags Reference

This document explains the labeling system used across Interwoven Projects.

Consistent labels are one of the main ways people discover relevant projects and contributions.

## Field Labels (`field:*`)

These indicate the primary disciplines a project touches.

Current core set:
- `field:ai-ml`
- `field:physics`
- `field:history`
- `field:philosophy`
- `field:coding-software`
- `field:biology`
- `field:writing`
- `field:design`
- `field:mathematics`
- `field:art-creative`
- `field:policy-governance`

**How to add new ones**: Open a simple issue or PR suggesting the label. We keep the list reasonably small but are happy to expand.

## Status Labels

- `status:seeking-contributors` — Open for new people to join
- `status:needs-lead` — Has energy but needs someone to coordinate
- `status:active` — Has a committed team and is moving
- `status:paused` — Temporarily on hold
- `status:completed` — Main work finished (credit recorded)

## Needs Labels (`needs:*`)

Used to signal specific help required right now.

Examples:
- `needs:developer`
- `needs:writer`
- `needs:domain-expert`
- `needs:researcher`
- `needs:designer`
- `needs:reviewer`
- `needs:data-experiments`
- `needs:facilitator`
- `needs:connector`

## Other Useful Labels

- `beginner-friendly` — Good entry point for newer community members
- `credit-open` — Used on Contribution Update issues
- `good-first-issue` — For improvements to the hub repository itself
- `proposal` — Applied to new Project Proposal issues (can be added automatically via forms)
- `discussion` — For meta or community process threads

## GitHub Discussions Categories

We also organize conversation with categories (set in repo settings):

- Project Proposals
- Introductions & Skills
- Help Wanted / Looking for Collaborators
- Show & Tell / Completed Work
- General / Meta

## Best Practices

- Apply 1–3 field labels + 1 status label to most proposals.
- Update labels as projects evolve (especially status).
- When in doubt, use broader labels rather than inventing very narrow ones.
