Each subfolder represents one active (or recently completed) project.

## How to Add a New Active Project

1. When a proposal has a small team and clear momentum, create a new folder here:
   ```
   projects/YYYY-MM-short-kebab-name/
   ```
2. Copy the template from `templates/project-readme-template.md`
3. Fill it out with current status, contributors, and links. **Note:** The Project Lead must be an approved Moderator.
4. Update this table.
5. Move the corresponding card on the GitHub Projects board to "Active".

## Project Folder Contents (Recommended)

- `README.md` — Main project page (use the template)
- `CONTRIBUTORS.md` — Living credit record (highly recommended)
- Any lightweight shared assets, notes, or links to external repos

For larger projects (especially those with code, data, or many files), we strongly encourage spinning up a dedicated repository under the Interwoven Projects organization (once created) and linking back here.
